Перед запуском тестов нужно проверить настройки в config.py
Тесты запускаются через start_all_tests.py
В start_all_tests.py есть список тестов запускаемых по очереди, при необходимости можно их поменять местами или дополнить
Сначала тесты проходят в Chrome потом при отсутствии ошибок в Firefox
После устанвки зависимостей, нужно выполнить playwright install
